Why you should use jenesius-vue-modal?

Preface

While developing websites using Vue 3, I began to encounter the inconvenience of using modal windows. Some kind of a smart guy a couple of years ago suggested opening them directly, embedding components and displaying them using v-if. But how inconvenient it is and clutters up the components. For me personally, this was incredibly inconvenient and I decided to write something of its own. And then, a couple of years later, there was a transition from Vue 2 to Vue 3 and an active library appeared.

Advantages

  • A large set of methods for opening/closing modal windows.
  • Ability to open several elements at once.
  • A set of hooks for tracking the closing of windows.
  • Integration into vue-router.
  • A large number of examples of use.
  • Refined documentation.

If you are interested, go to the documentation site.
